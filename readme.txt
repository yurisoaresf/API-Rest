Projeto para criação de serviços web - Rest, utilizando as seguintes tecnologias:


MongoDB - Banco de dados NoSQL

Express - Framework utilizado pra criação de um servidor web, assim como, poder importar todas as funções e métodos necessários para integração de API Rest, com todos os métodos HTTP, etc.

Nodemon - Ferramenta (Lembra o LiveServer) que faz com que toda modificação feita no projeto, seja atualizada in real time, sem precisar parar a aplicação e rodar novamente, com o "npm start" por exemplo.

Mongoose - Framework estilo JPA, só que para Node.js, ou seja, ele tem todos os métodos CRUD para voce realizar através de uma IDE de programação conectando com um SGBD externo, neste caso, com o MongoDB.

Body-Parser - Framework que transforma todo dado que vem de frontend, em objetos(req.body).
