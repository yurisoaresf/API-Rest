Este Readme apenas relata o que foi usado neste projeto, como:


MongoDB - Banco de dados NoSQL

Express - Framework utilizado pra criaçao de um servidor web, assim como, poder importar todas as funções e métodos necessários para integração de API Rest, com todos os metodos HTTP, etc.

Nodemon - Espécie de plugin (Lembra o LiveServer) que faz com que toda modificação feita no projeto, seja atualizada em tempo real, sem precisar parar a aplicação e rodar novamente com o npm start por exemplo.

Mongoose - Framework estilo JPA, só que para Node.js, ou seja, ele tem todos os métodos CRUD para voce realizar com o banco MongoDB.

Body-Parser - Transforma todo dado que vem de frontend, em objetos(req.body).
